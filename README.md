# Resume Maker Web App

A simple web application for creating and printing resumes.

## Features

- Live preview of resume as you type
- Responsive design
- Print functionality
- Sections for personal information, education, work experience, and skills

## Usage

1. Open `index.html` in your web browser
2. Fill in the form with your resume details
3. See the live preview update in real-time
4. Click "Print Resume" to print or save as PDF

## Files

- `index.html`: Main HTML structure
- `styles.css`: Styling for the application
- `script.js`: JavaScript for functionality
- `README.md`: This file
